# SnapShotPro 架构分析

## 项目简介

**SnapShotPro** 是一个轻量级 Windows 截图工具，使用 C# + WinForms 开发，目标是单文件可执行程序（~15MB），兼容 Windows 7 SP1 及以上系统。

---

## 主要功能

### 截图模式
- 区域截图（拖拽选区）
- 窗口截图（点击选择窗口）
- 全屏截图
- 延时截图（2s / 5s）
- 全局热键触发（默认 Alt+T / Alt+E / Alt+F）

### 标注工具
- 箭头、矩形、文字、马赛克（隐私打码）
- 撤销/重做（最多 20 步）
- 颜色选择

### 输出方式
- 保存为文件（PNG / JPG / BMP，自动时间戳命名）
- 复制到剪贴板
- 自动复制开关

### 系统集成
- 常驻托盘运行
- INI 配置文件持久化
- 热键可自定义，冲突检测

## 模块说明

### Core — 系统集成层

| 文件 | 职责 |
|------|------|
| ScreenCapture.cs | BitBlt 屏幕捕获；DWM 真实窗口边界；多显示器支持；PrintWindow 回退 |
| WindowDetector.cs | EnumWindows 枚举窗口；按坐标查找窗口；窗口层级导航 |
| HotkeyManager.cs | Win32 RegisterHotKey 封装；全局热键注册与管理 |
| Hotkey.cs | 热键配置序列化（修饰键 + 主键）；字符串解析与显示 |
| ClipboardHelper.cs | 图像写入剪贴板 |

### UI — 用户界面层

| 文件 | 职责 |
|------|------|
| SelectionOverlay.cs | 全屏半透明覆盖层；实时选区预览；窗口悬停高亮 |
| EditorForm.cs | 标注编辑器主窗口；工具栏；状态栏；快捷键支持 |
| TrayApp.cs | 托盘图标与右键菜单；热键协调；延时截图计时器 |
| SettingsForm.cs | 热键自定义对话框；重复热键验证 |
| HotkeyRecorder.cs | 专用 TextBox，捕获键盘输入并实时显示修饰键组合 |

### Annotation — 标注引擎

| 文件 | 职责 |
|------|------|
| AnnotationCanvas.cs | 双缓冲画布；原图层 + 标注层；工具工厂；撤销集成 |
| ITool.cs | 工具接口（OnMouseDown/Move/Up, DrawPreview）+ ToolType 枚举 |
| AnnotationHistory.cs | 撤销/重做栈（最多 20 步）；克隆状态保存 |
| ArrowTool.cs | 三角函数计算箭头方向与翼点 |
| RectangleTool.cs | GDI+ 矩形；坐标归一化支持任意拖拽方向 |
| TextTool.cs | 文字渲染（含阴影）；TextInputDialog 获取用户输入 |
| MosaicTool.cs | 12px 块均值算法；区域隐私打码 |

### Utils — 工具类

| 文件 | 职责 |
|------|------|
| Settings.cs | INI 配置读写；启动时加载；热键/路径/格式默认值 |
| FileHelper.cs | 保存/另存为对话框；时间戳文件名；格式检测；目录创建 |

## 架构分层与数据流

```
Program.cs → TrayApp（托盘控制器）
    ↓ 热键触发
SelectionOverlay（全屏选区覆盖层）
    ↓ 用户完成选区
ScreenCapture（BitBlt / DWM / PrintWindow）
    ↓ 返回 Bitmap
EditorForm（标注编辑器）
    ↓ 用户标注完成
AnnotationCanvas + Tools（双缓冲画布 + 工具链）
    ↓
FileHelper / ClipboardHelper（导出）
```

**完整流程：**
1. 用户按下全局热键（Alt+T / Alt+E / Alt+F）
2. HotkeyManager 触发回调 → TrayApp 调度
3. SelectionOverlay 显示全屏覆盖层，用户选区或点击窗口
4. ScreenCapture 执行截图（BitBlt 为主，DWM/PrintWindow 为辅）
5. EditorForm 加载截图，用户使用工具标注
6. 用户保存（FileHelper）或复制（ClipboardHelper）

## 技术亮点

| 特性 | 实现方式 |
|------|---------|
| 兼容性 | .NET Framework 4.0，支持 Windows 7 SP1+ |
| DPI 感知 | app.manifest 声明，避免 Win7 坐标缩放问题 |
| 真实窗口边界 | DWM Extended Frame Bounds API，去除 Win10/11 阴影 |
| 最小化窗口截图 | PrintWindow + PW_RENDERFULLCONTENT；全黑检测自动回退 |
| 双缓冲渲染 | 原图层与标注层分离，导出时合并，避免破坏原图 |
| 自排除检测 | SelectionOverlay 预缓存窗口列表后显示自身，防止自检测 |
| 热键冲突防护 | 注册失败显示气泡提示，不崩溃；UI 层提前验证重复 |

## 配置文件（SnapShotPro.ini）

```ini
SaveFolder=        # 截图保存目录
ImageFormat=png    # 图片格式：png / jpg / bmp
AutoCopy=0         # 截图后自动复制到剪贴板：0/1
HotkeyRegion=Alt+T     # 区域截图热键
HotkeyWindow=Alt+E     # 窗口截图热键
HotkeyFullscreen=Alt+F # 全屏截图热键
```
